是双层板 3D打印主板-stm32高性能迷你8轴主板 的升级版本

正式起名为: 小螃蟹主板

核心板和底板均为四层板

工程来源变更原理等请看旧版工程描述

 

核心板:

核心板GPIO保护升级, 除can和USB端子无GPIO保护外, 其余引脚也有GPIO保护, IO过压会触发底板自动关机

核心板mcu依然采用stm32h750VBT6, usb转串口芯片换成了ch343p, CAN通信芯片换成了TJA1051T/3

核心板叠了两层插件端子,需要先焊接背面的端子, 把端子的引脚剪短不露头再焊接

核心板的USB/CAN/串口, 无需调整线序对接  3D打印CAN工具头-stm32f446小螃蟹满配工具头

1. 2个3pin端子, 可以用于限位, 5V供电输出

2.  4个2pin温控端子, 可以用于限位

3.  4个4pin端子,  1个ch343p uart接口, 1个usb接口, 1个can接口, 1个I2C接口

4. 1个USB typeA母座

 

底板:

底板布局优化, 砍掉了驱动跳线选择改为背面焊接跳线选择UART/SPI/普通模式, step1,step2改成了独立EN

如果使用旧版本的核心板,需要飞一条线来使能step1的EN

底板单片机stc32g8k64 5V控制引脚改动, 避免开机时5V电源闪烁一下

底板的35V电容都换成了插件电容, 容易焊接, 容量更大

 

1.  6个限位端子, STOP1和STOP2供电为12V/24V切换输出, IO有二极管隔离, 可以高压输入, STOP3,STOP4 为2PIN限位端子,与STEP1/STEP2驱动的无限位归零共用, STOP5,STOP6为3PIN限位端子5V供电输出

2. 4个可控风扇端子, FAN1,FAN2为5V/12V/24V切换供电, FAN3,FAN4为24V固定供电

3. 8个步进电机驱动, 前6个支持UART/SPI/普通模式, 后两个只支持UART模式

4. 2个大功率加热棒, 需要单独供电, 需要焊接引出

5. 电源监控上位机页面地址不变, 线上访问地址: https://demo.chengxg.top/mini8/

 

MCU引脚定义

[board_pins mcu_pins]
mcu: mcu # stm32h750 V2核心板
aliases:
    # step1-step6支持spi, spi_bus: spi2
    DIR1=PD3, STEP1=PD4, UART1=PD2, EN1=PD6,   # step1  
    DIR2=PC8, STEP2=PC12, UART2=PD0, , EN2=PD1,          # step2
    DIR3=PC9, STEP3=PD8, UART3=PD14, EN3=PB1,  # step3
    DIR4=PC6, STEP4=PD13, UART4=PD12,          # step4 EN与step3共用
    DIR5=PE14, STEP5=PD15, UART5=PD9, EN5=PB2, # step5
    DIR6=PC5, STEP6=PB12, UART6=PB0,           # step6 EN与step5共用
    DIR7=PA5, STEP7=PA6, UART7=PA7, EN7=PC4,   # step7
    DIR8=PA2, STEP8=PA3, UART8=PA1, EN8=PA0,   # step8
    FAN1=PA8, FAN2=PC7, FAN3=PA10, FAN4=PA9,   # FAN1, FAN2支持5V/12V/24V电压切换
    STOP1=PA4, STOP2=PE12,            # 限位 3PIN STOP1,STOP2 支持高压输入,二极管输入隔离,支持12V和24V供电输出,
    STOP3=PE13, STOP4=PE11,           # 限位 2PIN step1,step2无限位归零
    STOP5=PE10, STOP6=PE9,            # 限位 3PIN 5V供电输出
    STOP7=PB7, STOP8=PB6,             # 核心板上, 限位 5V供电输出
    T1=PC0, T2=PC3, T3=PC1, T4=PC2,   # 核心板上, 温控
    H1=PD11, H2=PD10,                 # 加热 H1大功率
    LED=PB4,                          # klipper运行指示 !PB4
    SDA=PB11, SCL=PB10,               # 核心板上, I2C接口 i2c_bus:i2c2
    SHUT=PD7                          # 高电平 自动关机
